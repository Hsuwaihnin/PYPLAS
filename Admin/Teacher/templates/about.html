How To Use PYPLAS

PYPLAS is used to test programming skill.
So, this system is used by teacher and students.


Teacher
Teachers use PYPLAS to test student's programming skill.
Firstly, teachers write or copy program code from textbook and other python code.
In PYPLAS,teachers choose python code for question.
Teacher press the question button to give question program code automatically.
Then, this questions save in database automatically.
Teacher edit or delete question.
Teacher look the result of student answers.

Students
Students use PYPLAS to test student's programming skill.
Firstly, students look the questions program and write a correct answers.
After students had answered one question program then they press the 'submit' button.
So, students look their result and reanswer the questions program until they get correct answers.
Then, students press the 'save' button to save the answers code in database and
to test the student programming skill automatically.
